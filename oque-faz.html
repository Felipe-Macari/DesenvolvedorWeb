O que faz um Desenvolvedor Web?

Front-end: cuida da parte visual do site — o que o usuário vê e interage. Usa HTML, CSS e JavaScript para criar layouts e interfaces.
Back-end: trabalha na parte interna do site — lógica, segurança e banco de dados. Faz o sistema funcionar por trás das telas, usando linguagens como Python, Node.js, PHP, Java etc.
Full-stack: une as duas áreas — sabe criar tanto o visual (front) quanto o funcionamento interno (back). É o profissional mais completo do desenvolvimento web.
